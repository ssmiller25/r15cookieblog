# CONVENTIONS.md

Portable conventions for terminal-first agents (Aider CLI and similar tools).

- Treat AGENTS.md as the canonical instruction file.
- Select one lead persona from AGENTS.md before making edits.
- Keep diffs small, reversible, and scoped to the request.
- Run the smallest relevant validation command before finalizing.
- Escalate cross-domain conflicts using AGENTS.md handoff rules.
